# silang-api

## Setup
Set up your virtual environment
### Windows
```bash
python -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
```
### Linux or MacOS
```bash
python -m venv env
source ./env/bin/activate
pip install -r requirements.txt
```

## Running the app
`fastapi dev src/main.py`
